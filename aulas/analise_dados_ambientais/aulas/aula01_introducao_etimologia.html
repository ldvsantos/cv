INTRODUÇÃO

ETIMOLOGIA

Definição:

Inferência estatística refere-se aos resultados derivados da análise estatística dos dados coletados.

Ou seja, a inferência estatística advém de uma relação entre a teoria estatística e os dados reais.

CONJUNTO DE DADOS, POPULAÇÃO E AMOSTRA

Conjunto de dados:

Coleção de observações onde cada coluna representa uma variável e cada linha uma unidade de análise. Cada valor é um item de dado.

População:

Conjunto completo de elementos que compartilham uma característica; representa o universo do estudo.

Amostra:

Subconjunto da população selecionado para representar o todo, utilizado quando o acesso à população é impraticável.

TIPOS DE VARIÁVEIS

De acordo com Stevens (1946), as variáveis podem ser classificadas como:

Normal

Ordinal

Escalar

Razão

O tipo de variável determina a análise a ser utilizada

VARIÁVEIS NOMINAIS

Uma variável totalmente qualitativa

  • Não significa que não pode ser feita análises quantitativas Os valores apenas diferenciam categorias qualitativas diferentes, mas não há relação entre as categorias.

Ex: nacionalidade, religião, estado civil

Os números atribuídos não tem significado matemático.

  • Brasileiro
  • Argentino
  • Chinês

VARIÁVEIS ORDINAIS

Há uma diferença entre as categorias

  • Estabelece uma ordem ou hierarquia entre as categorias.
  • No entanto, o intervalo entre as categorias não é regular. A ordem de chegada dos corredores na linha de chegada é um exemplo de medida ordinal, Você sabe quem chegou primeiro, mas o dado por si só não informa se a diferença do segundo foi de um segundo ou uma hora

VARIÁVEIS ESCALARES E DE RAZÃO

O intervalado entre os valores é sempre constante.

O que diferencia a variável escalar e razão é o significado do valor 0.

Zero da escala intervalar é um valor arbitrário.

0º C;

A nota de um teste de matemática pode ser uma medida intervalar:

Cada unidade da nota corresponde a uma questão correta.

Entretanto, uma nota zero não corresponde a ausência de conhecimento matemático.

DISTRIBUIÇÃO DE FREQUÊNCIAS

Conceito:

Uma distribuição de frequências organiza os dados em categorias ou intervalos e mostra quantas observações pertencem a cada classe.

Configura um caso particular das séries estatísticas, uma vez que os fatores época, local e fenômeno permanecem fixos, o fenômeno apresenta-se através de “gradações”.

Utilizada para resumir toda a informação obtida, de modo a facilitar a análise.
