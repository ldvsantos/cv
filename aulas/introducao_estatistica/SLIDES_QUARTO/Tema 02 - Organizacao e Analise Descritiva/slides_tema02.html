I. Introdução

3.1 — Normas para Tabelas

Uma tabela compõe-se de:

  • Corpo — conjunto de linhas e colunas com informações sobre a variável
  • Cabeçalho — parte superior, especifica o conteúdo das colunas
  • Coluna indicadora — especifica o conteúdo das linhas
  • Linhas — retas imaginárias que facilitam a leitura horizontal
  • Casa ou célula — espaço destinado a um só número
  • Título — Conjunto de informações respondendo: O quê? Quando? Onde?

Normas IBGE para Tabelas

De acordo com as normas do IBGE, nas casas ou células devemos colocar:

Símbolo Significado
— (traço horizontal) Valor é zero
… (três pontos) Não temos dados
? (interrogação) Dúvida à exatidão do valor
0 (zero) Valor muito pequeno para ser expresso pela unidade utilizada

Normas APA (7ª edição)

De acordo com o Publication Manual of the APA:

  • Valor zero → utilizar 0 (zero)
  • Dado indisponível → utilizar NA (not available)
  • Valor duvidoso → utilizar (?) com nota de rodapé
  • Valores muito pequenos → notação científica (ex: \(3{,}45 \times 10^{-5}\))

Observação Importante

Na APA, nunca se deve deixar células vazias em tabelas.

Sempre que símbolos, abreviações ou convenções não forem autoexplicativas, deve haver uma nota de tabela explicando-os.

3.2 — Séries Estatísticas

Séries Estatísticas

Consistem na apresentação das informações em forma de tabelas, objetivando sintetizar os dados observados.

Uma série estatística é uma sucessão de valores expressando dados estatísticos referidos a qualquer variável.

Tipos de Séries Estatísticas:

  1. Histórica, Cronológica ou Temporal
  2. Geográfica ou de Localização
  3. Específica, por categoria ou especificativa
  4. Mista (ou tabela cruzada)

1. Série Histórica, Cronológica ou Temporal

Os dados são observados segundo a época de ocorrência. O elemento variável é a época (tempo).

Exemplo: Tabela 6 – Evolução da relação candidato/vaga (2009–2012) do curso de Geografia da UEFS

Ano Relação candidato/vaga
2009 1,76
2010 6,65
2011 2,81
2012 3,46

Fonte: Dados fictícios

2. Série Geográfica ou de Localização

Apresenta como elemento variável o fator geográfico. Dados observados segundo a localidade.

Exemplo: Tabela 7 – População urbana por região, em 1990, no Brasil

Região Pop. urbana (%)
Norte (N) 54,8
Nordeste (NE) 57,2
Sul (S) 78,7
Sudeste (SE) 90,1
Centro-Oeste (CO) 79,2

Fonte: IBGE

3. Série Específica ou por Categoria

Os dados são agrupados segundo a modalidade de ocorrência.

Exemplo: Tabela 8 – Nº de acidentes de trânsito por tipo de veículo em Feira de Santana (Set/2021)

Tipo de veículo Nº de acidentes
Automóvel de passeio 243
Caminhão 107
Ônibus 83
Motocicleta 472
Total 905

Fonte: Dados Fictícios

4. Série Mista (Tabela Cruzada)

É a combinação de um ou mais tipos de séries descritas anteriormente.

Exemplo: Tabela 9 – Produção Agrícola por região, segundo os principais produtos – 1994 (toneladas)

Região Arroz Feijão Milho Total
Norte (N) 1,5 45,2 5,0 51,7
Nordeste (NE) 2,4 120,0 1,6 124,0
Sul (S) 12,5 75,0 15,5 103,0
Sudeste (SE) 30,7 100,0 10,3 141,0
Centro-Oeste (CO) 10,5 10,5 4,0 25,0

Fonte: Dados Fictícios

Exercício

Pesquisar em jornais, revistas e/ou publicações digitais, exemplos de séries estatísticas:

  1. Histórica

  2. Geográfica

  3. Específica

  4. Mista

Para exercitar a identificação do tipo de série. Aproveite a oportunidade e averigue os títulos das tabelas e quadros.

# Instalar pacote (se ainda não tiver)
install.packages("knitr")
library(knitr)

# Criar data frames com exemplos de séries estatísticas
historica <- data.frame(
  Ano = c(2010, 2015, 2020, 2023),
  IPC = c(5.9, 10.7, 4.5, 5.6)
)

geografica <- data.frame(
  Regiao = c("Norte", "Nordeste", "Sul", "Sudeste", "Centro-Oeste"),
  IDH = c(0.667, 0.663, 0.756, 0.766, 0.757)
)

kable(historica, caption = "Série Histórica: IPC por Ano")
kable(geografica, caption = "Série Geográfica: IDH por Região")
